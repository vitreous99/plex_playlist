# Security Policy

If you've found a security vulnerability in this project, thank you — we take these seriously.

Please report security issues via GitHub's security advisories (recommended):

- Use the "Security" → "Report a vulnerability" flow on GitHub for this repository.

For non-sensitive questions, you can also open an issue and mark it as private.

Response targets:
- Acknowledge receipt: within 48 hours
- Initial triage: within 5 business days

We will coordinate disclosure and provide fixes as appropriate.

Note: Do not include exploit code or sensitive data in public issues.
